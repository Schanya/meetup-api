## Meetup-api
CRUD REST Web API for organizing of meetups

Web API functionality:
1. Getting a list of all meetups;
2. Getting a certain meetup by its Id;
3. Registration of a new meetup;
4. Changing information about an existing meetup;
5. Removing the meetup.

Meetup's information:
1. Title / topic;
2. Description;
3. A set of tags / keywords;
4. Time and place of the event.

Technology stack:
1. Node.js;
2. Nest.js;
3. PostgreSQL 14;
4. Sequelize.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
